# Legacy Strapi v4 Archive

The old Strapi v4 app and plugin are kept outside this course repo to keep the Strapi 5 reboot clean.

Archive repo:

- `https://github.com/newt-build/strapi-v4-course-archive`

Contents:

- `app`: archived Strapi v4 course app, formerly `strapi-devblog`.
- `plugin`: archived Strapi v4 plugin, formerly `strapi-plugin-github-projects`.

This code is reference-only and unsupported. New lessons, checkpoints and support material should point to the Strapi 5 code in this monorepo.
